# Superhero TTRPG Character Sheet

Interactive superhero TTRPG character sheet built as a single HTML file.

## Features

- Editable character and hero identity fields
- Portrait, full-body character, and full hero costume image upload slots
- Two parent name fields with parent photo upload slots
- Appearance details for blood type, eye color, hair, and skin
- Separate character and hero costume description fields
- Quirk description with selectable quirk speciality
- Rank, karma, walking movement, flying movement, current health, and max health
- Damage and healing controls with a live health bar
- Ability scores including Charisma and Perceptive, statistics sliders, and radar chart
- Dice roller
- Quirk, traits, tags, biography, and eight numbered quirk cards
- Save/load in the browser
- Separate saved sheets with `?character=CharacterName` in the URL
- Import/export sheet data as JSON
- Printable layout
